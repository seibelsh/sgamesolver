Defining games
==============

In sGameSolver, stochastic games can be defined using the
:ref:`SGame <sgame>` class.

Defining a stochastic game requires three inputs:

1. payoffs
2. transitions
3. discount factors

These can either be submitted as arrays or as Excel-like tables.
